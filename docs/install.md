# Install guide

1. Install Java
2. Install Maven
3. Install pip
4. Install pyenv
5. Install autoenv
6. Install postgres
7. Install python 3.8 using pyenv `pyenv install 3.8.0`
8. Create virtual-env with python 3.8 `pyenv virtualenv 3.8.0 shankara`
9. Install requirements `pip install -r requirements.txt`
